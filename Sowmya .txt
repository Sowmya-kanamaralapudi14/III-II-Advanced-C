3.Question: Write a Python program to reverse a number.
Answer:
Python
Copy code
num = int(input("Enter number: "))
rev = 0


while num > 0:
    digit = num % 10
    rev = rev * 10 + digit
    num = num // 10


print("Reverse:", rev)




4.Write a Python program to check whether a number is palindrome or not.
Answer:
Python
Copy code
num = int(input("Enter number: "))
temp = num
rev = 0


while num > 0:
    digit = num % 10
    rev = rev * 10 + digit
    num = num // 10


if temp == rev:
    print("Palindrome")
else:
    print("Not Palindrome")




5.Write a Python program to find the factorial of a number.
Answer:
Python
Copy code
num = int(input("Enter number: "))
fact = 1


for i in range(1, num + 1):
    fact = fact * i


print("Factorial:", fact)

8.Write a Python program to check whether a number is prime.
Answer:
Python
Copy code
num = int(input("Enter number: "))
flag = 0


for i in range(2, num):
    if num % i == 0:
        flag = 1
        break


if flag == 0:
    print("Prime number")
else:
    print("Not prime")
